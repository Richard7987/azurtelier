Cada cubo modular es una unidad autónoma diseñada para proporcionar un ambiente controlado y óptimo para el cultivo de hongos. A continuación, se describen sus elementos clave:

## 1. 🧱 Estructura del Cubo  

- **🔧 Material**:  
  - Plástico resistente o acrílico (ligero, duradero y aislante).  

- **📐 Tamaño estimado**:  
  - Entre **30 y 40 cm por lado**.  

- **🧊 Aislamiento térmico**:  
  - El cubo contará con una doble pared con **material aislante** (tipo baúl) para conservar la temperatura interna, especialmente útil si se integran sistemas de calefacción o enfriamiento.


## 2. 🔌 Componentes Internos  

### 🔍 Sensores  
- **💧 Sensor de humedad** (DHT22 o similar):  
  - Mide y regula la humedad mediante el control del humidificador.  
- **🌡️ Sensor de temperatura** (DHT22 o DS18B20):  
  - Detecta cambios térmicos y activa la calefacción si es necesario.  
- **🫁 Sensor de CO₂** (MH-Z19 o similar):  
  - Monitoriza los niveles de dióxido de carbono para asegurar una correcta aireación.

### ⚙️ Actuadores  
- **💨 Humidificador compacto**:  
  - Pequeño, silencioso y automático según los datos del sensor de humedad.  
- **🔥 Elemento calefactor (resistencia o manta térmica)**:  
  - Calienta el interior si la temperatura baja del umbral definido.  
- **🌬️ Ventilador de bajo consumo**:  
  - Favorece el intercambio de aire y evita zonas de humedad estancada.

### 🎥 Cámara (opcional)  
- **Cámara con visión infrarroja**:  
  - Conectada a un sistema de IA que analiza el crecimiento, color y tamaño del hongo mediante modelos de *machine learning* para predecir el momento óptimo de cosecha.


## 3. 🌐 Conectividad y Energía  

### 📡 Conexión a red  
- **Módulo Wi-Fi (ESP32 o similar)**:  
  - Para enviar datos de sensores a la central mediante **MQTT**, **HTTP** o **WebSockets**.  

### 🔋 Alimentación  
- **Puerto de energía individual**:  
  - Cada cubo se conecta al estante para alimentación y respaldo energético.  
- **Fuente regulada**:  
  - Adaptador con protección térmica y contra sobrecargas.


## 4. 🧩 Accesibilidad y Modularidad  

### 🛠️ Sistema de montaje  
- **Guías o rieles**:  
  - Para que los cubos puedan deslizarse fácilmente dentro y fuera del estante vertical.  

### 🔗 Conectividad rápida  
- **Puerto estándar de conexión** (USB o pines magnéticos):  
  - Permite integrar rápidamente alimentación y comunicación con el sistema central.


> ✅ Un diseño modular y estandarizado facilitará la escalabilidad del sistema y el mantenimiento de cada unidad.
